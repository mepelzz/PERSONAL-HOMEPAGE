PROJECT 01
📦

Desktop Application

DataBoost Inventori

Aplikasi desktop berbasis Java untuk manajemen inventori barang secara lengkap. Dibangun dengan arsitektur MVC dan terhubung ke database MySQL, mendukung multi-role pengguna (Admin & Staf).

JavaJavaFX / SwingMySQLMVC

Fitur Utama

▹Dashboard monitoring stok dengan grafik real-time dan notifikasi stok kritis
▹Modul transaksi penjualan dengan kalkulasi otomatis dan riwayat history
▹CRUD data inventori lengkap dengan filter kategori dan pencarian barang
▹Manajemen pengguna multi-role dengan autentikasi login yang aman
▹Laporan PDF otomatis untuk stok, penjualan, dan pembelian
PROJECT 02
✊

AI / Computer Vision

Gesture Lock System

Sistem keamanan berbasis gestur tangan menggunakan Rock Paper Scissors sebagai "password" visual. Model KNN dilatih dengan hand landmark MediaPipe, akurasi 100% pada data uji.

PythonMediaPipeOpenCVScikit-learnKNN

Fitur Utama

▹Deteksi 21 hand landmark secara real-time menggunakan MediaPipe
▹Klasifikasi 3 gesture (Batu, Gunting, Kertas) dengan model KNN
▹Sistem lock/unlock berdasarkan urutan kombinasi gestur sebagai password
▹Akurasi model 100% pada data uji (Confusion Matrix KNN)
▹Mendeteksi gesture UNKNOWN untuk input yang tidak dikenali
